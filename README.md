## Standards

Chaque projet de cette galerie est :

- **Déployé** — une URL live existe, pas juste du code
- **Documenté** — README avec description, stack, démo, limites connues
- **Sobre** — stack proportionnelle au problème résolu
- **Validé** — testé par au moins un utilisateur réel

Lire le [manifeste complet](manifesto.md) pour comprendre ce qui mérite d'être ici.
